# 专注提醒器

使用笔记本摄像头，在本机识别离开画面、持续转头、手机和部分常见玩物，并可对正脸或侧脸保持基本不动约30秒的状态给出“疑似发呆”提示。

## 使用方法

1. 将笔记本放在书桌正前方或斜前方，接通电源。
2. 双击 `启动专注提醒.bat`。服务会在后台运行，不需要保留黑色窗口。
3. Microsoft Edge 打开后，点击“启动摄像头”，选择“允许”。
4. 孩子保持正常写作业坐姿，点击“校准正面方向”。
5. 建议先由家长陪同测试，再调整提醒时间和灵敏度。

开启“持续提醒”后，分心状态未恢复时会每约 6 秒重复播放所设次数的提示音；恢复正常或离开画面后停止。

## 隐私

- 摄像头画面只在浏览器内存中处理。
- 软件不录像、不截图、不上传视频。
- 提醒记录只保存在当前页面，刷新后清空。

## 判断限制

软件不能准确区分发呆、思考或看题，只能依据可观察行为给出“疑似”提示；笔等小物件也可能无法识别。

视觉模型与浏览器运行库来自 Google MediaPipe Tasks Vision。
